  # プライバシーポリシー

  「やめドラ」（以下、本アプリ）は、ユーザーのプライバシー保護を最優先に設計されています。本ポリシーでは、本アプリが取得・
  利用する情報の範囲を明示します。

  ## 1. 取得しない情報

  本アプリは、以下の情報を**一切取得しません**。

  - 氏名・住所・電話番号・メールアドレスなどの個人識別情報
  - 位置情報
  - 連絡先・カレンダー・写真ライブラリ
  - カメラ・マイクへのアクセス
  - アカウント情報（ログイン機能はありません）
  - 健康・フィットネス情報、メッセージ、音声、ファイル

  なお、広告配信の目的で **広告 ID（Android Advertising ID / iOS IDFA）** が AdMob SDK
  によって取得・使用されます。詳細は「4. 広告について」を参照してください。

  ## 2. 端末内に保存される情報

  ユーザー体験のために、以下の情報を**端末内のローカルデータベース（Isar）にのみ**保存します。これらの情報がインターネット 
  経由で外部サーバーに送信されることはありません。

  - キャラクターの成長状況（種類・進化段階・お世話の累積ポイント）
  - 月間負け額の設定値（節約額の計算に使用）
  - 「行きたくなった」記録の発生時刻と乗り切り結果
  - アプリの開封日時（オフライン進行の計算に使用）
  - マイルストーン達成記録（30日 / 100日 / 300日）

  データはアプリのアンインストールと同時に端末から完全に削除されます。

  ## 3. 通信について

  本アプリは、以下の限られた目的のためにインターネット通信を行うことがあります。

  - **フォント取得**: Google Fonts（DotGothic16）の初回取得
  - **外部リンクの起動**: 設定画面の「相談先」リンクから、厚生労働省・GA Japan などの**外部 Web
  サイトをユーザー操作で開く**場合
  - **シェア機能**: 「自慢する」操作で生成した画像とテキストを、ユーザーが選択した SNS / メッセンジャーアプリに OS
  のシェアシート経由で渡す場合
  - **広告配信**: AdMob SDK による広告のロードと表示（「4. 広告について」参照）

  これらはいずれもユーザーの明示的な操作・選択を伴うか、広告表示のための限定された通信です。本アプリ自身が独自にユーザーの 
  行動を収集・送信することはありません。

  ## 4. 広告について

  本アプリは Google の **AdMob** を利用してバナー広告を配信しています。

  ### 4-1. 広告で利用する情報

  AdMob SDK は広告配信のため、以下の情報を取得・Google に送信します。

  - **広告 ID（Android Advertising ID / iOS IDFA）**: 広告配信のパーソナライズおよび効果測定のために使用されます。
  - 端末情報（OS バージョン、画面サイズなど、広告フォーマット選択のために必要な最小限の情報）

  これらは AdMob SDK によって暗号化通信（HTTPS）で Google
  に送信されます。本アプリのデベロッパー（Melt-dev）が直接これらの情報を取得・保存することはありません。

  ### 4-2. 広告 ID のオフ / リセット方法

  ユーザーは端末の OS 設定により、広告 ID をいつでもリセット・オフにすることができます。

  - **Android**: 設定 → Google → 広告 → **「広告 ID をリセット」**または**「広告 ID を削除」**
  - **iOS**: 設定 → プライバシーとセキュリティ → トラッキング → 「App からのトラッキング要求を許可」を OFF（または ATT     
  ダイアログで「Appにトラッキングしないように要求」を選択）

  ### 4-3. 広告カテゴリの除外

  本アプリは依存症対策アプリであるため、以下の広告カテゴリを AdMob 管理画面側で**ブロック設定**しています。

  - ギャンブル全般（カジノ・スロット・賭博）
  - 飲料アルコール
  - デート / 出会い系

  加えて、コード側でも
  `MaxAdContentRating.g`（一般視聴者向け）を設定し、暴力・性的・成人向けコンテンツの広告を排除しています。

  リワード広告 / 動画広告 / インタースティシャル広告（全画面割込み広告）は使用しません。バナー広告のみで、ホーム画面下端と 
  設定画面下端に配置されます。アプリ内のキャラ育成 / ケアポイントと広告視聴は一切結びつきません。

  ### 4-4. 詳細

  AdMob のプライバシー詳細については、以下の Google 公式ポリシーをご参照ください。

  - Google プライバシーポリシー: <https://policies.google.com/privacy>
  - AdMob プライバシーセンター: <https://support.google.com/admob/answer/6128543>

  ## 5. 第三者ライブラリの利用

  本アプリは、以下のオープンソースライブラリを利用しています。これらは本アプリ自体がデータを収集する手段として用いるもので 
  はありません。

  - Flutter / Dart SDK（Google）
  - flutter_riverpod, isar, go_router, share_plus, url_launcher, intl, google_fonts ほか
  - **google_mobile_ads（Google）**: 広告配信のため。本ポリシー「4. 広告について」参照
  - 各ライブラリのライセンス条文はアプリ内ライセンス画面（将来追加予定）または本リポジトリの `pubspec.yaml`
  経由で確認できます

  ## 6. 子どもの利用について

  本アプリは依存症対策の自助ツールであり、未成年者を主な対象とはしていません。
  13 歳未満のユーザーから個人情報を取得することはありません（取得自体が無いため）。

  ## 7. 医療行為の代替ではないこと

  本アプリは医療行為や治療の代替ではありません。本アプリの利用がギャンブル依存症の治療効果を保証するものではなく、症状の改 
  善・治癒を断定的に示すものでもありません。
  医療機関での治療を受けている方は、主治医の指示を優先してください。
  本アプリの結果や使用に関連する一切の損害について、開発者は責任を負いません。

  ## 8. ポリシーの変更

  本ポリシーは、機能変更や法令対応のため予告なく更新されることがあります。
  重要な変更がある場合はアプリ内および本ページにて告知します。

  ## 9. お問い合わせ

  本ポリシーに関するお問い合わせは、以下までお願いします。
